



## Analysis Methods

| Acronym | Full Name | Description |
|---------|-----------|-------------|
| FTA | Fault Tree Analysis | Top-down deductive failure analysis method |
| TCA | Total Cause Analysis | Structured root cause analysis approach |
| RCA | Root Cause Analysis | General term for identifying the underlying cause of a failure |

## AI / Data

| Acronym | Full Name | Description |
|---------|-----------|-------------|
| RAG | Retrieval-Augmented Generation | LLM pattern: retrieve relevant context, then generate an answer |
| LLM | Large Language Model | AI model used for text generation and extraction (e.g. Claude) |
| KG | Knowledge Graph | Graph-structured representation of entities and their relations |

## Ontology / Semantic Web

| Acronym | Full Name | Description |
|---------|-----------|-------------|
| RDF | Resource Description Framework | W3C standard for representing graph data as triples |
| RDFS | RDF Schema | Basic class/property hierarchy layer on top of RDF |
| OWL | Web Ontology Language | Rich ontology standard built on RDF, supports logical reasoning |
| SKOS | Simple Knowledge Organization System | RDF vocabulary for thesauri and controlled vocabularies |
| SHACL | Shapes Constraint Language | RDF validation rule language |

## Data Source (FKD site)

| Acronym | Full Name | Description |
|---------|-----------|-------------|
| FKD | Failure Knowledge Database | 失敗知識データベース — Japanese database of industrial failure cases |
| ASF | Association for the Study of Failure | NPO that maintains the FKD |
| HA | Hazard Analysis file | Full-text PDF report per case (`HA*.pdf`) |
| CA | Case Analysis file | HTML case page per case (`CA*.html`) |
| SA | Summary Analysis file | HTML summary page per case (`SA*.html`) |
| MA | Media file | Images associated with a case (`MA*.jpg/gif`) |
